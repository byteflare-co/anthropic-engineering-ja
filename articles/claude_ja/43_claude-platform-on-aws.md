# Claude Platform on AWS のご紹介

Claude Platform on AWS が本日より一般提供（GA）となり、AWS のお客様は AWS の認証、請求、コミットメントの消化のもとで、Claude Platform のすべての機能にアクセスできる新しい方法を手に入れました。Claude は引き続き Amazon Bedrock 上でも利用可能で、こちらは AWS がデータプロセッサーとなります。

本日より、Claude Platform on AWS のお客様は [Claude Managed Agents](https://claude.com/blog/claude-managed-agents) を使ってエージェントを大規模にデプロイし、コード実行、Skills、アドバイザー戦略（advisor strategy）などのツールを使った開発が可能になります。

## AWS 経由で Claude Platform のフル機能にアクセスする

Claude Platform on AWS は、AWS のお客様に対して Claude API の全機能を初めて提供します。新機能やベータ機能はすべて、ネイティブの Claude API でローンチされたその日のうちに同様に提供されます。

認証は AWS IAM 経由、監査ログは CloudTrail 経由、請求はひとつの AWS インボイス経由で行われ、既存のコミットメントに対して完全に消化されます。お客様は既存の AWS の認証情報と IAM ポリシーを利用するため、チームは既に管理しているツールと権限のなかにとどまったまま作業できます。

Claude Platform on AWS は AWS のほとんどの商用リージョンで利用可能となる予定で、グローバルおよび米国の推論ジオグラフィをサポートします。
