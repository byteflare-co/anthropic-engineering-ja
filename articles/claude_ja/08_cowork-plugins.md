# プラグインで Cowork をカスタマイズ

Cowork では、あなたがゴールを設定すれば、Claude が完成度の高いプロフェッショナルな成果物を提供します。プラグインを使えばさらに一歩進められます。仕事の進め方、参照すべきツールやデータ、重要なワークフローの処理方法、チームに公開するスラッシュコマンドなどを Claude に指示することで、より良く、より一貫性のあるアウトプットが得られます。

## プラグインで Cowork を使う

プラグインはあらゆるユースケースに対応しますが、営業、法務、財務分析など、特定の職種に合わせて Claude をカスタマイズする場合に特に威力を発揮します。

たとえば営業プラグインは、Claude を CRM やナレッジベースに接続し、営業プロセスを教え、見込み客のリサーチからコールのフォローアップまで、あらゆることに対応するコマンドを提供できます。プラグインに何を入れるかを一度定義すれば、Claude は関連する場面でそのコンテキストを自動的に活用します。

![](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/697bffa5de78fa874a5efed8_Hero%20Image%20-%20plugins.png)

チームがプラグインを構築・共有するにつれて、Claude は部門横断的なエキスパートになっていきます。共有されたリッチなコンテキストは関連するすべてのやり取りに組み込まれるため、リーダーや管理者はプロセスの徹底に費やす時間を減らし、プロセスの改善により多くの時間を使えるようになります。

![](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/697bfe022fd5ecc76e2b5a3b_c546a845.png)

## プラグインマーケットプレイス

すぐに使い始められるよう、私たち自身のチームで構築・使用している 11 個のプラグインをオープンソースで公開します:

- **Productivity** -- タスク、カレンダー、日々のワークフロー、個人のコンテキストを管理
- **Enterprise search** -- 社内のツールやドキュメント全体から情報を検索
- **Plugin Create/Customize** -- 新しいプラグインをゼロから作成・カスタマイズ
- **Sales** -- 見込み客のリサーチ、案件の準備、営業プロセスに沿った対応
- **Finance** -- 財務分析、モデル構築、主要指標の追跡
- **Data** -- データセットのクエリ、可視化、解釈
- **Legal** -- ドキュメントのレビュー、リスクのフラグ付け、コンプライアンスの追跡
- **Marketing** -- コンテンツのドラフト、キャンペーンの計画、ローンチの管理
- **Customer support** -- 問題のトリアージ、返信のドラフト、ソリューションの提示
- **Product management** -- 仕様書の作成、ロードマップの優先順位付け、進捗の追跡
- **Biology research** -- 文献検索、結果の分析、実験の計画

これらは Cowork から直接インストールできるほか、[ウェブサイト](https://claude.com/plugins-for/cowork)でコレクション全体を閲覧したり、独自のプラグインをアップロードしたりすることもできます（Plugin Create を使って構築可能）。開発者向けには [GitHub](https://github.com/anthropics/knowledge-work-plugins) でも公開しています。プラグインのすべてのコンポーネント（Skills、コネクタ、スラッシュコマンド、サブエージェント）はファイルベースなので、構築、編集、共有が簡単です。Claude を使って各プラグインをカスタマイズし、あなたとチームにぴったりのものに仕上げてみてください。

## 始め方

Cowork のプラグインサポートは、すべての有料 Claude ユーザー向けにリサーチプレビューとして本日から利用可能です。オープンソースのコレクションから始めて、カスタマイズするか、まったく新しいものを構築してください。

プラグインは現在、ローカルマシンに保存されます。組織全体での共有・管理のサポート（プライベートプラグインマーケットプレイスなど）は、今後数週間のうちに提供予定です。
