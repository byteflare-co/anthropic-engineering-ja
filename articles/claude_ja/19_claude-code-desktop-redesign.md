# 並列エージェントのための Claude Code デスクトップ刷新

複数のセッションを管理するための新しいサイドバー、ワークスペースを自在に配置できるドラッグ&ドロップのレイアウト、統合ターミナルとファイルエディタ、そしてパフォーマンスと使い勝手の向上を盛り込みました。

## 新しいデスクトップ体験

多くの開発者にとって、エージェント的な作業のかたちは変わりつつあります。プロンプトを 1 つ打ち込んで待つ、というスタイルではありません。あるリポジトリでリファクタリングを、別のリポジトリでバグ修正を、さらにもう 1 つでテスト作成のパスを走らせながら、結果が返ってくるたびに各セッションを確認し、ずれたら軌道修正し、出荷前に差分をレビューする——そういう働き方になっています。

新しいアプリは、エージェントコーディングの今の手触り、つまり多くの作業が同時進行していて、あなたが指揮者の席に座る、という状況に合わせて作られています。

## セッションを並列で走らせる

新しいサイドバーには、アクティブなセッションと最近のセッションがすべて一か所にまとまっています。複数のリポジトリにまたがって作業を開始し、結果が返ってくるタイミングでそれらの間を行き来できます。

ステータス、プロジェクト、環境でフィルタリングしたり、サイドバーをプロジェクトごとにグループ化することで、セッションを素早く見つけて再開できます。セッションの PR がマージまたはクローズされると、そのセッションは自動的にアーカイブされるので、サイドバーは現在進行中のものに集中した状態を保てます。

作業中に何かを質問したくなったら、サイドチャット（⌘ + ; または Ctrl + ;）を開いて会話を分岐させられます。サイドチャットはメインスレッドからコンテキストを引き継ぎますが、メインスレッドに何かを書き戻すことはありません。こうすることで、タスクの方向を誤らせずに済みます。

## アプリから出ずにレビューして出荷する

今回のリデザインでは、よく使うツールをアプリ内に取り込み、エディタに行き来することなく Claude の成果物をレビュー・調整・出荷できるようにしました：

- **統合ターミナル**: セッションと並行してテストやビルドを実行できます。
- **アプリ内ファイルエディタ**: ファイルを開き、その場でちょっとした編集を加え、変更を保存できます。
- **高速化された差分ビューア**: 大きな変更セットでもパフォーマンスが出るように再構築されました。
- **拡張されたプレビュー**: プレビューペインでローカルアプリのサーバーを動かすのに加えて、HTML ファイルや PDF をアプリ内で開けます。

すべてのペインはドラッグ&ドロップに対応しています。ターミナル、プレビュー、差分ビューア、チャットを、自分の働き方に合わせて好きなグリッドに並べてください。

## 既存のスタックにフィットする

デスクトップアプリは CLI プラグインと同等の機能を備えるようになりました。組織が Claude Code プラグインを集中管理している場合でも、あなた自身がローカルにインストールしている場合でも、ターミナルでの挙動とまったく同じようにデスクトップアプリでも動作します。

セッションはローカルでもクラウドでも引き続き実行できます。SSH サポートが Linux に加えて Mac にも拡張されたので、どちらのプラットフォームからでもセッションをリモートマシンに向けられます。

## 自分の働き方に合わせてカスタマイズする

Verbose、Normal、Summary の 3 つの表示モードによって、Claude のツール呼び出しまで完全に見せるインターフェースから、結果だけを見せるインターフェースまで、ダイヤルで調整できます。セッションの切り替え、立ち上げ、ナビゲーションに対応する新しいキーボードショートカットを追加しました。一覧は `⌘ + /`（または `Ctrl + /`）で確認できます。新しい使用量ボタンでは、コンテキストウィンドウとセッションの使用状況の両方をひと目で確認できます。

内部的にも、アプリは信頼性と速度のために再構築され、Claude が生成する応答をストリーミング表示するようになりました。

## はじめ方

リデザインされたデスクトップアプリは、Pro、Max、Team、Enterprise プランのすべての Claude Code ユーザー、および Claude API 経由で、現在ご利用いただけます。

[アプリをダウンロード](https://claude.com/download)するか、すでにインストール済みの場合はアップデートして再起動してください。詳しくは[ドキュメント](https://docs.claude.com/claude-code)をご覧ください。
