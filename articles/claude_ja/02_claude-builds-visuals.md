# Claude がインタラクティブなチャート・図・ビジュアライゼーションを作れるようになりました

Claude の会話にはすでに[アーティファクト](https://claude.ai/redirect/claudedotcom.v1.db7e93a1-fc67-4391-8723-ea2ac51deb65/catalog/artifacts)があります——Claude が作成する永続的なツールやドキュメントで、より洗練された成果物として共有・ダウンロードできるように設計されています。これに対し、今回のチャート・図・ビジュアライゼーションは別の目的を果たします——Claude がいま議論しているトピックに対するユーザーの理解を助けるために作るものです。それらはサイドパネルではなく本文中にインラインで現れ、一時的なものとして、会話が進むにつれて変化したり消えたりします。

いくつかの例を挙げましょう。Claude に「複利はどう働くのか?」と尋ねると、操作して遊べる曲線を返してくれます。あるいは周期表について尋ねれば、クリックして詳細を掘り下げられるインタラクティブなビジュアライゼーションを作ってくれます——以下の例のように。

![](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/69b1048a4e146f048af72e00_Claude-Blog-Inline-Visualization.gif)

この機能はデフォルトで有効になります。Claude は何かに対してビジュアルを作るべきタイミングを自分で判断しますし、ユーザーが直接リクエストすることもできます (「図として描いて」「時間とともにどう変化するか可視化して」といった問いかけで)。Claude が何かを作成したあとで、調整を依頼したり、さらに深掘りを頼んだりできます。

これらのビジュアルは、私たちが最近 Claude の応答に加えてきたより広範な改善の一部です。今年の初め、Claude は特定のトピックに対して目的特化型のフォーマットを使い始めました——たとえばレシピは今や材料と手順付きで表示され、天気について聞けば Claude はビジュアルを提供してくれます。議論の中で Figma、Canva、Slack のようなアプリと直接[やり取り](https://claude.com/blog/interactive-tools-in-claude)することもできます。

![](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/69b1fc1487a93a1dd7c2a174_Claude-Blog-InlineVisuals-2.png)

今日からお試しいただけます。この機能はすべてのプランタイプで利用可能です。
