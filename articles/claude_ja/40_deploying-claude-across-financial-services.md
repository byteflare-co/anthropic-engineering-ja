# 金融サービス全体に Claude をデプロイする

金融サービス業界では、リサーチ、ディール業務、アンダーライティング、保険金請求対応、モデルレビュー、月次決算など、さまざまな現場で Claude が稼働しています。そして、そのほとんどの企業が複数の Claude プロダクトを組み合わせて使っています。チャットとリサーチには Claude を、ファイルやアプリをまたぐプロジェクトレベルの業務には Claude Cowork を、クオンツやエンジニアリングのチームには Claude Code を、Excel や PowerPoint、Word、Outlook で過ごす時間には Claude for Microsoft 365 を、そしてカスタムアプリケーションやエージェントの構築・デプロイには Claude Platform と Claude Managed Agents を、それぞれ活用しています。

リーダーがロールアウトの道筋を描けるよう、金融サービス向けの実践的なデプロイメントガイドをまとめました。各プロダクトの解説に加え、Anthropic がリリース済みのファイナンスエージェント向けリファレンスアーキテクチャと、各社が今日から実行できる 3 段階の導入プレイブックを取り上げています。

このガイドでは以下を共有しています。

- Claude Chat、Cowork、Code、Microsoft 365 アドイン、Claude Platform、Claude Managed Agents をいつ使うかを示すプロダクトマトリクス
- ピッチ作成、市場リサーチ、KYC スクリーニング、総勘定元帳の照合、月次決算など、10 種類の事前構築済みファイナンスエージェントテンプレート
- AIG、Commonwealth Bank of Australia、IG Group、Moody's のお客様事例
- 各段階で機能するアクションとプロのコツをまとめた、3 段階（基盤、パイロット、スケール）の導入プレイブック
- 投資銀行とプライベートエクイティ、ウェルスマネジメントとアセットマネジメント、リテールバンキングとコマーシャルバンキングなど、セクター別に整理されたユースケース

このガイドは、自社のどこに Claude が当てはまるかを判断する AI リーダーと、それをロールアウトするエンジニアの双方に向けて作られています。
